Crypto

A study in cryptographic encryption and decryption algorithms

My first CLI project with python; this project implements several encryption algorithms, along with timing features, used to estimate success of brute force attacks with sequential processors.
The program can handle encoding and decoding DES, 3DES and AES ciphers, with functions for symmetric key generation, in addition to an RSA encoder/decoder, with asymmetric key generation to text files also possible.
The interface is accessible on the right, and this project can be downloaded from my github page.

ChessBot

Creating a bot that beats me in chess (shouldnt be too hard)

For this project, I created a chess game engine that can play games with human vs human, human vs bot, or bot vs bot.

I thought it would be fun to attempt to create a fleet of bots that are each of varying difficulties, using completely different algorithms instead of just adjusting the depth of the same minimax algorithm.
I cant beat the most advanced bot, so I consider this project to be a huge success.

Djikstra

An implementation of djikstra's algorithm to find optimal routes

This repository began with a vision close to google maps, but this will do.

Amusingly, the hardest thing to implement in this project was just getting the little numbers to sit next to the arcs within the canvas, I feel like it required the invention of a whole new algorithm.
Until told that it already exists, or that there is a better algorithm for it, I shall coin this function-for-putting-little-numbers-next-to-lines-in-2d, 'The Angus Algorithm'.

TerminalGames

I made this so I could play games during history lessons

While this project was initially not for serious development, when I began extending the games, i found that making 2 player games required opponents that would actually be fun to play against.

Noughts and Crosses introduced alpha-beta pruning in minimax trees, and battleships opened a tin of statistical analysis that was mind numbing.
But on the plus side I've not won at battleships in a long time, so a success in my eyes.
