Wordle

A simple Wordle clone

This project is a clone of the wordle game in which a player needs to guess the secret 5 letter word.

After each guess the colours of the letters indicate their existance and position in the secret word.



Gray - letter doesn't exist in the word

Yellow - letter exists in a different location

Green - letter exists in that exact location



The solution button is not a part of the game, it's existance just makes it easier to see how the colouring system works (i.e. how multiple letters are handled etc.)
